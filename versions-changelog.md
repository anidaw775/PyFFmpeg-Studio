# Changelog

All notable changes to this project will be documented in this file.

## v1.1
#### 2026/06/01
### Added

#### Video

* Batch processing (multiple files at once).
* Video preview before conversion.
* Rotate video by 90°/180°.
* Playback speed adjustment.
* Add a watermark (text or image).
* Add subtitles.
* Change FPS.
* Normalize audio volume in videos.

#### Audio

* Trim audio.
* Adjust volume.
* Merge multiple audio files.

#### Convenience

* Task queue.
* Operation history.
* Save settings profiles.
* Dark and light themes.
* FFmpeg log in a separate window.
* Estimated completion time display.

#### Advanced Features

* Codec selection (H.264, H.265, AV1, VP9).
* Hardware acceleration support:

  * NVIDIA NVENC
  * Intel Quick Sync
  * AMD AMF
* Custom FFmpeg parameters.
* File information display (codec, bitrate, FPS, resolution).

#### Special Features

* Download videos from YouTube using yt-dlp.
* Create timelapse videos.
* Thumbnail generator.
* Automatic compression to a target file size.
